### **MurirTin - A Smart Service of Unbranded Buses**

#### **How to Run The Project:**

* **Transfer all files in a dedicated folder**
* **Download and install Android studio**
* **Open that folder in android studio**
* **Go to app/kotlin+java/com.example.murirtin**
* **Give some time to sync all gradle files**
* **Add an emulator or use an android phone to run the app**
* **Hit the run button and the app will run** 



#### **Files Details:**

* **DataModel.kt has all the classes used in this project**
* **AuthRepository.kt has all functions that access database**

#### 

#### **API Details:**

* **Firebase api**
* **app/kotlin+java/res/values/string.xml has the api key for Google Map AI (the free trial will end on 9 January 2026)**
